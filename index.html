About

Most teenage projects die by month three.

Most teenage projects die by month three. Mine stick. I'm drawn to long games — the kind of work that only gets interesting after you've shown up for it a few hundred times in a row. Starting is easy; I'm more interested in what survives the part where it stops being fun.

In practice that looks like a daily podcast I've run for years, a karate-school app that real instructors actually use, and a four-year book drive. I like building software, figuring out how businesses actually work, and turning both into things people use. If I start it, I plan to be doing it years from now.
